# 🥐 Bakery Website

A modern bakery website built as one of my first assignments during the **Frontend Development Diploma at Route Academy**.

The project was initially built as a **native desktop layout**, and later I enhanced it by making the website **fully responsive** and adding **animations** to improve the user experience.

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Responsive Design
- CSS Animations
- Modern Layout Techniques
